## 🌐 How to Host on GitHub Pages
1. Go to https://github.com/new
2. Create a new public repo (e.g., anniversary-page)
3. Upload all files from this ZIP
4. Go to Settings → Pages → Build and deployment → select `main` branch → Save
5. Wait 1–2 minutes, then your site will be live at:
   `https://<yourusername>.github.io/anniversary-page/`
